# Changelog

All notable changes to this project are documented in this file.

## [1.0.0] - 2026-07-17

### Added

- Schema and row-level validation with traceable rejection reasons
- Accepted and rejected order datasets
- Gross and recognized revenue calculations using integer cents
- Customer, product, category, and monthly sales summaries
- Reconciliation checks across cleaned orders and every summary table
- Atomic output staging with preservation of the previous complete report set
- Run identifiers, timestamps, durations, quality rates, and operational logs
- Strict data-quality mode and defined command exit codes
- Ruff, pytest, coverage enforcement, and GitHub Actions checks
